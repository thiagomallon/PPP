PSkelG¶

Classe implementa comando de criação de classe de teste, pelo phpunit skeleton generator, porém, via php cli, para maior agilidade na criação de skeletons de teste. Para essa classe, as questões de namespace e path foram feitas de forma que atendem exclusivamente às necessidades desse projeto, sendo ignorada a necessidade evidente de um adapter, para tornar as especificações mais generalizadas, ou até o recebimento de mais parâmetros pela função.

Comando gerado: phpunit-skelgen generate-test --bootstrap="path/bootstrap.php" "namespace\of\class" path/of/class/Class.php "namespace\of\testclass\classtest" path/of/testclass/ClassTest.php
